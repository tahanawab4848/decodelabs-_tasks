# DecodeLabs Full Stack Development Internship 🚀

Welcome to my portfolio repository for the **DecodeLabs Full Stack Development Internship**. This repository contains a progressive series of projects built to master modern full-stack web development, specifically focused on creating a **Cricket Club Management System**.

The journey is divided into four main projects, progressing from a static frontend UI all the way to a fully authenticated, database-driven MERN stack application.

---

## 📂 Repository Structure & Projects

### 1️⃣ [Project 1: Frontend UI Design](./Cricket_Project-P1)
The foundational stage. A completely static frontend built from scratch with semantic HTML5 and vanilla CSS3 to establish the visual language (warm 2025 aesthetics) of the Cricket Club Management System.

### 2️⃣ [Project 2: Backend API with In-Memory Storage](./CricketAPI-P2)
The backend foundation. A robust RESTful API built with Node.js and Express.js featuring JWT-based authentication, strict data validation, and custom rate limiting. Data is temporarily managed in-memory to focus on API architecture and routing.

### 3️⃣ [Project 3: Database Integration](./CricketMemory-P3)
The data persistence layer. This project upgrades Project 2 by integrating a real MongoDB database via Mongoose. It introduces complex data models, relationships (like Players and Coaches), strict database-level validation, and advanced query filtering.

### 4️⃣ [Project 4: Complete Full-Stack Integration](./CricketComplete-P4)
The mastery phase. This brings everything together by connecting the refined UI from Project 1 to the secure, database-driven backend from Project 3. It utilizes the Fetch API, async/await patterns, dynamic DOM manipulation, and token-based state management to create a seamless Single Page Application (SPA) experience.

---

## 🛠️ Technology Stack

**Frontend:**
- HTML5 & CSS3 (Mobile-first responsive design)
- Vanilla JavaScript (ES6+, DOM Manipulation, Fetch API)

**Backend:**
- Node.js & Express.js
- MongoDB & Mongoose ODM
- JWT (JSON Web Tokens) for authentication
- bcrypt for password hashing
- express-validator & express-rate-limit
