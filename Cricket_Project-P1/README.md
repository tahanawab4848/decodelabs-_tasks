# Project 1: Frontend UI Design 🎨

This is the first project of the DecodeLabs Full Stack Development Internship. The goal of this phase is to build a responsive, visually appealing frontend for a **Cricket Club Management System** using purely HTML5 and CSS3.

## 🎯 Objectives
- Establish the visual identity using a warm 2025 aesthetic color palette.
- Implement a mobile-first responsive design using modern CSS features (Flexbox/Grid).
- Structure the application using clean, semantic HTML5 tags for accessibility and SEO.
- Create the foundational UI components (Navbars, Cards, Modals, Forms) that will be integrated with the backend in later phases.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Vanilla)
- Google Fonts (Inter & Roboto)

## 🚀 How to Run
This is a static website. You do not need any servers or dependencies to run it.
1. Clone or download the repository.
2. Navigate to the `Cricket_Project-P1` folder.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

## 📱 Features
- Responsive Navigation Bar
- Hero Section with clear call-to-actions
- Player Roster Grid System
- Reusable button and form styling
